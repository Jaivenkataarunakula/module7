HTTP Status Codes

Name Numerical Code Purpose Situation
Unprocessable Entity 422 The server understands the content type of the request entity, and the syntax of the request entity is correct, but it was unable to process the contained instructions. Often returned when the server cannot process a POST or PUT request due to semantic errors.
Too Many Requests 429 The user has sent too many requests in a given amount of time ('rate limiting'). Returned when a user exceeds the rate limits defined by the server.
Bad Gateway 502 The server was acting as a gateway or proxy and received an invalid response from the upstream server. Returned when the proxy server gets an invalid response from an external server.
Service Unavailable 503 The server is not ready to handle the request. Common causes are a server that is down for maintenance or that is overloaded. Returned during high traffic events when the server is unable to handle the request due to being overloaded.
Gateway Timeout 504 The server was acting as a gateway or proxy and did not receive a timely response from the upstream server. Returned when a request times out at the proxy server because the upstream server did not respond in time.
